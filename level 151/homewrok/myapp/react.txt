React არის JavaScript ბიბლიოთეკა, რომელიც გამოიყენება ui-ს შესაქმნელად. 
ის საშუალებას გვაძლევს სწრაფად და ეფექტურად შევქმნათ აპლიკაციები კომპონენტების მეშვეობით.

React გამოიყენება ვებ-აპლიკაციების შესაქმნელად, სადაც ინფორმაცია ხშირად იცვლება. 
იგი ამარტივებს UI-ს შექმნას, ხელახლა გამოყენებად კომპონენტებს ქმნის და უზრუნველყოფს სწრაფ მუშაობას.

JSX არის JavaScript-ის გაფართოება, რომელიც გვაძლევს საშუალებას HTML კოდი პირდაპირ ჩავწეროთ JavaScript-ში. 
React-ში JSX გამოიყენება კომპონენტების UI-ს აღწერისთვის უფრო მარტივი და მკაფიო ფორმით.
